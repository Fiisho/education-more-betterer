Data, Types, & Structures Distinguish between Data Types and Data Structures
Questions Progress
Data Types are Types of data, compared to its brethren Data Structures which are, you guessed it, Structures for data. Data Types relate to the format of data and how it's classified, where as Data Structures refer to the collection of data that is organised so it can be executed more effciently. Data Types refer to the collection of different kinds of data which can be represented using an object, however Data Structures refer to the form/kind of a variable being used throughout a program.
